# WebCodeDemo
该demo实现一个简单的前端模版（Vite+Vue3ts+Router+Pinia）
src目录构成如下
WEBCODEDEMO\SRC      
├─assets            静态资源文件
├─components        Vue组件目录
├─layout            自定义布局元素
├─router            Vue Router相关配置
├─store             Pinia状态管理相关目录
│  └─modules
├─typings           TypeScript的类型定义文件
├─utils             工具函数的目录
└─views             Vue视图组件目录
    ├─home
    └─login

UI像关组件推荐Element-plus、less、Tailwind CSS
